# Airtbnb Clone Project

에어비앤비와 같은 최신 스타트업들에게 많이 요구되는 기술 (예약. 후기. 메시징. 어드민 패널)을 처음부터 끝까지 직접 만들어본 풀스택 개발자가 된다.

# Concept & Function

---

\***\*Users\*\***

- User Authentication
- Kakao Talk Login
- Github Login
- Admin Panel
- Host / Admin Mode
- Direct Messages

\***\*Rooms\*\***

- Room CRUD
- Reservation Calendar
- Room Admin
- Image Upload
- Review System

\***\*Deployment\*\***

- AWS Elastic Beanstalk
- AWS S3
- AWS RDS
